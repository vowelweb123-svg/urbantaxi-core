Complete Elementor widget suite for UrbanTaxi theme.

== Description ==

UrbanTaxi Core provides 12 professional Elementor widgets in a single unified plugin. All widgets include full Elementor integration, responsive design, and production-ready code.

**Included Widgets:**
- Heading Style
- Booking Slider
- Cab Filter
- Client Testimonial
- Book Seat
- Our Blog
- Service Cards
- Smart Animations
- Sticky Mission
- Taxonomy
- Team Carousel
- Timeline

== Installation ==

1. Activate the plugin through WordPress admin
2. Go to Elementor editor to use widgets
